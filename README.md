# Trabajo Final - Piedra, Papel o Tijera Avatar

## Descripción

Este proyecto consiste en el desarrollo de un videojuego de **Piedra, Papel o Tijera** con temática de **Avatar**, implementado en lenguaje C utilizando la biblioteca **raylib** para la interfaz gráfica.

El juego permite al usuario interactuar mediante un menú principal, iniciar partidas, configurar opciones y enfrentarse a la computadora en partidas de Piedra, Papel o Tijera.

El proyecto fue desarrollado siguiendo una arquitectura modular, utilizando múltiples ramas de Git para el desarrollo colaborativo.

---

## Características

- Menú principal interactivo.
- Interfaz gráfica desarrollada con raylib.
- Música y efectos de sonido.
- Sistema de Piedra, Papel o Tijera.
- Pantalla de opciones.
- Organización modular del código.
- Uso de Git y GitHub para control de versiones.

---

## Compilación

```bash
make
```

## Ejecución

```bash
./main
```
